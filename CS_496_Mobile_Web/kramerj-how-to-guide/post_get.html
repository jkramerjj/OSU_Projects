Making the API Work

Alright, so now we have OAUTH running within our Twitter application.

Now it's time we use the Twitter API for POST and GET. We want to POST a tweet and then GET a list of most recent tweets.

 

POST

As we have seen, we can start by declaring our objects and instantiating our classes.       

Twitter newTwitter = new TwitterFactory().getInstance();

Now I'm going to assume you have got a tweet string form the user and placed it into a variable called NewTweet. If this isn't making sense I suggest reading this basic Android Development Tutorial to get a better understanding of how to use TextView and XML. Or I suggest reading the Google App Engine Tutorial to get better undestanding of HTML forms.

OK, so now we'll go ahead and take our newTwitter object and run the .updateStatus() method, while passing in our newTweet string.

newTwitter.updateStatus(newTweet);

That's it. You have just sent a POST with your new Tweet to Twitter.

 

GET

As before we can start by declaring our objects and instantiating our classes.       

Twitter newTwitter = new TwitterFactory().getInstance();

Next we take our newTwitter object and we run the .getUserTimeline() method. 

newTwitter.getUserTimeline()

That's it. With those two lines of code you can GET an entire JSON list of all your tweets. From there you can place the tweets into a list array and print it to screen.

 

Here's a final bonus, because I know printing a Twitter JSON has it's own set of challenges:

protected void onPostExecute(List<twitter4j.Status> statusHolder){
    for(twitter4j.Status statsPlace : statusHolder){
        twitter_array.add(statsPlace.getText());
    }
    ArrayAdapter<String> adapterArray = new ArrayAdapter<String>(TwitterList.this, android.R.layout.simple_list_item_1, twitter_array);
    list_of_twitter.setAdapter(adapterArray);
}

The trick is declaring a ListView and a new ArrayList() before running the code above

private ListView list_of_twitter;
private List<String> twitter_array = new ArrayList<String>();
