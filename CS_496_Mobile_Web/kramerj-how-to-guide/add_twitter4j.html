How to add Twitter4J

What is Twitter4j? Twitter4J is an unofficial Java library for the Twitter API. Which is great, because they offer built-in OAUTH, POST, and GET support. If you'd like to learn more please visit their website.

OK so now we've found the library and we know what it can do, but how do we add it and how do we utilize it?

Now I'm going to give you a hint, that in my humble opinion makes this tutorial worth the read right now.  You see in Android Studio there is no need to download an entire Java library, then attempt to load and compile it correctly. What you will do is:

Open your builde.gradle file from the App directory and add this line of code:

dependencies {
compile 'org.twitter4j:twitter4j-core:4.0.2'
}

For fear that the read will leave this tutorial, I will not say that this little problem took me over a day to figure out. I will definitely not admit that...

 

OK well how about downloading the actual library and using the physical files. Well that's an option too and it's fairly straight forward. Here is what you'll want to do.

Go to the Twitter4J homepage and look for a download icon similar to this:





Now you need to unzip the files and place them into your projects' root directory.
