Purpose for the guide

The tutorial is dedicated to helping the novice user learn Node.js. First we focus on how to install correctly and then we work on the basics, leading up to coding asynchronously. The tutorial will take place in the LINUX/UNIX command line environment. This guide is designed to start with the very basics and to make "you" the reader feel comfortable navigating the complex and rewarding world of node.js. Are you worried that you might not understand node.js? Even after researching it? Well don't, because if you've ever worked with JavaScript you've basically worked with node.js. Here are a few reasons why learning node.js is a great idea!

  • It's very fast, because it uses the V8 engine developed by Google
    • Meaning all your script is compiled into machine code.
  • Use the same programming language between the server and the database and back to the user.
    • There is no need for translating between client data and server data.
  • Websockets run over TCP
    • This means less overhead as compared to using HTTP
  • Giant community
    • This is an active community with meet-ups, to open source, to dedicated IRC channels
  • It's written in the most popular web development language in the world.
  • JavaScript

What is Node.js?

Essentially it is an application written in Javascript that can be ran on systems from Windows, to OS X, to Linux, and more. It is an open source cross-platform runtime environment for networking and server applications. If anything just think of it as a "server-side interpreter" for JavaScript.

But what is one major benefit why I should consider Node.js?

Simplification. Instead of using PHP, you can now use Node.js to communicate with the database.
